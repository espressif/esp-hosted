# ESP-Hosted Coprocessor Examples

This directory contains example applications for the ESP-Hosted-FG coprocessor (ESP slave side). The examples are organized by host type:

## [MCU-based Examples](cp_mcu/README.md)
Examples for microcontroller-based hosts (STM32, etc.)
- Basic WiFi and Bluetooth functionality
- Custom RPC and Network Split features
- Detailed guide in [cp_mcu/README.md](cp_mcu/README.md)

## [Linux-based Examples](cp_linux_fg/README.md)
Examples for Linux-based hosts (Raspberry Pi, etc.)
- Basic WiFi and Bluetooth functionality
- Custom RPC and Network Split features
- Detailed guide in [cp_linux_fg/README.md](cp_linux_fg/README.md)

Choose the appropriate directory based on your host platform and refer to its README for detailed instructions.
